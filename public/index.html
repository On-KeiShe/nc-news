Northcoders NC News API

Welcome to the Northcoders News, a RESTful API providing access to a database of articles, topics, users and comments for a fictional news site as part of a backend project. All responses are in JSON format.

GET /api

This endpoint returns a JSON object describing all the available endpoints in the API, along with their methods and descriptions.

Route Parameter: None

Query Parameters: None

Request Body: Not required

Example Response:

GET /api/topics

This endpoint returns a list of all available topics on the site. Each topic groups related articles together.

Route Parameter: None

Query Parameters: None

Request Body: Not required

Example Response:

GET /api/articles

This endpoint returns a list of all articles. Articles can be filtered, sorted, and ordered using optional query parameters.

Route Parameter: None

Query Parameters:

  • Filter articles by topic
  • Sort articles by any valid column, e.g.created_at, votes, comment_count
  • Order by ascending or descending (default is descending

Request Body: Not required

Example Response:

PATCH /api/articles/:article_id

This endpoint allows you to update the vote count of a specific article by its article_id.

Route Parameter: The ID of the article you want to update (e.g./api/articles/1)

Query Parameters: None

Request Body: Required

Example Response:

GET /api/articles/:article_id/comments

This endpoint returns all the comments for a specific article based on its article_id.

Route Parameter:The ID of the article whose comments you want to retrieve (e.g. /api/articles/1/comments)

Query Parameters: None

Request Body: Not required

Example Response:

POST /api/articles/:article_id/comments

This endpoint allows you to add a new comment to a specific article.

Route Parameter: The ID of the article you want to comment on (e.g./api/articles/1/comments)

Query Parameters: None

Request Body:

Example Response:

DELETE /api/comments/:comment_id

This endpoint deletes a comment based on its comment ID.

Route Parameter: The ID of the comment you want to delete (e.g. /api/comments/12)

Query Parameters: None

Request Body: Not required

Example Response:
